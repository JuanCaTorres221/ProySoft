Proyecto de desarrollo de software enfocado en la creacion de un sistema de gestión de emociones.

UNA VEZ AHI COPIARÁ Y PEGARÁ LOS SIGUIENTES COMANDOS PARA PODER CORRER BIEN EL FRONT:

npm install react-router-dom
npm install react-icons @tailwindcss/forms
