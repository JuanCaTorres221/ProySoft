Proyecto de desarrollo de software enfocado en la creacion de un sistema de gestión de emociones.


NOTA IMPORTANTE, AL HACER GIT CLONE, IR AL CMD Y SELECCIONAR LA CARPETA DE FRONTEND.
UNA VEZ AHI COPIARÁ Y PEGARÁ LOS SIGUIENTES COMANDOS PARA PODER CORRER BIEN EL FRONT:


npm install

npm install react-router-dom

npm install react-icons @tailwindcss/forms

npm install recharts


NOTA: en el caso de que en "npm install" le pregunte descargar darle a (y)

Despues en la carpeta kairo/src/main/.../mainkairoapplication, ese archivo se corre

y en la terminal CMD nos ubicamos en la carpeta de frontend y ejecutamos "npm run dev"
